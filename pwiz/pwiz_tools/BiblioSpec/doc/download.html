There are several BiblioSpec packages available. The pre-compiled binary packages contain all of the BiblioSpec tools ready to run on your computer. Choose the package for your platform. The source code package contains the code and libraries required to build BiblioSpec from scratch. You will also need to download and build the proteowizard library. All packages contain html documentation and a README file with installation instructions.

  • Pre-compiled binaries for Linux x86_64
  • Pre-compiled binaries for Windows
  • Source files
  • proteowizard library (external library required for build)
