mspicture

mspicture is a command line tool for generating pseudo 2D gels from MS data files.

Examples:

  • mspicture
    (prints usage information)
  • mspicture data.mzXML
    (creates pseudo2D gel of data.mzXML)
  • mspicture --bry -p data.pep.xml data.mzXML
    (creates a blue-red-yellow color coded pseudo2D gel with pepXML annotations)
