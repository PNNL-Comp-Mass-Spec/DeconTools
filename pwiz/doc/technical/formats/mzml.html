What is mzML?

mzML is the new mass spectrometric data format, developed by the Mass Spectrometry Standards Working Group of the HUPO Proteomic Standards Initiative (HUPO-PSI).   mzML version 1.1 was released in June 2009.  

mzML was the combined effort of research centers and instrument vendors worldwide.   It was designed to be a replacement for the mzXML and mzData formats.

For more information about the status of the mzML format, see the HUPO-PSI mzML development page.

mzML and ProteoWizard

ProteoWizard provided the reference implementation of the mzML format during the review process. 

ProteoWizard includes a standard C++ platform-independent library for reading and writing mzML.  In addition, ProteoWizard provides CLI bindings for use from C# .NET applications.  ProteoWizard also provides a plug-in Reader interface to simplify conversions from vendor proprietary or other open formats to mzML. 

ProteoWizard has been released with the permissive Apache v2 license, which allows usage in both commercial and non-commercial projects, in order to encourage adoption of the mzML standard.
