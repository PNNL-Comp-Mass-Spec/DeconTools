ProteoWizard Installation and Data Formats

Installation

To install the ProteoWizard tools, download the binary package for your platform, and unzip it into the directory of your choice.

For convenience, adjust your "PATH" environment variable to include your ProteoWizard tools directory, so that the command line tools may be run from any directory.

Open Formats

ProteoWizard supports the reading/writing of the following open formats on all platforms:

  • mzML
  • mzXML
  • MGF
  • MS2/CMS2/BMS2
  • mzIdentML

Proprietary Formats supported by ProteoWizard

ProteoWizard depends on vendor-supplied software libraries to access data in proprietary formats.

Common requirements:

  • Windows XP, Vista, or 7
  • MSVC 2008 SP1 Redistributable Package (x86) installed
  • .NET Framework 3.5 SP1 installed

You can check to see whether the required components are installed:

  • (Windows XP) Control Panel->Add/Remove Programs
  • (Windows Vista/7) Control Panel->Programs and Features
