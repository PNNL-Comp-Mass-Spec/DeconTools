ProteoWizard mzXML information

What is mzXML?

mzXML is a mass spectrometric data format, developed by Institute for Systems Biology.   It has been deprecated in favor of the new mzML format.  

mzXML and ProteoWizard

ProteoWizard provides read/write access to mzXML.   For more information, see the ProteoWizard mzML page.
