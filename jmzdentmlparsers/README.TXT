1- To test Omssa2mzid converter:
Double click on omssa.bat


2- To test Tandem2mzid converter:
Double click on tandem.bat

3- To run Omssa2mzid converter:
java -jar "MzidParsers.jar" Omssa2mzid 55merge_omssa.omx 55merge_omssa.mzid

Or you can simply change your input file (55merge_omssa.omx) and your output file (55merge_omssa.mzid).


4- To run Tandem2mzid converter:
java -jar "MzidParsers.jar" Tandem2mzid 55merge_tandem.xml 55merge_tandem.mzid

Or you can simply change your input file (55merge_tandem.xml) and your output file (55merge_tandem.mzid).
